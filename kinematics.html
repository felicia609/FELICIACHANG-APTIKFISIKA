Kinematics

Definition

Kinematics is the study of how things move. It explains speed, distance, and direction without talking about forces. It helps us understand motion in everyday life, like a car driving or a ball falling.

Distance and Displacement

Distance: The total path length traveled by an object, regardless of direction. It is a scalar quantity.

Displacement: The shortest straight-line distance from initial to final position, including direction. It is a vector quantity.

Speed and Velocity

Speed: Rate at which an object covers distance.

Speed = Distance / Time

Velocity: Rate at which an object changes position in a direction.

Velocity = Displacement / Time

Acceleration

The rate at which velocity changes over time.

Acceleration = Change in Velocity / Time

Positive acceleration means speeding up, while negative acceleration (deceleration) means slowing down.

Average Speed

Total distance traveled divided by total time taken.

Average Speed = Total Distance / Total Time

Distance - Time Graph

  • The slope represents the speed of the object.
  • Straight line: Constant speed.
  • Curved line: Changing speed (acceleration or deceleration).
  • Horizontal line: The object is at rest.
  • Steep slope: Fast movement.
  • Shallow slope: Slow movement.
